# Glossari

**Rider** — Document amb necessitats tècniques d’un artista.

**Contra-rider** — Document de resposta per part de la sala/productor.

**Seal / Segell** — Marca digital incrustada al PDF que valida el rider.

**IA heurística** — Regles locals barates que analitzen el rider.

**IA avançada** — Anàlisi amb model LLM (pagament).

**R2** — Object Storage de Cloudflare.

**Patchlist** — Llista d’entrades de l’escenari.

**Stage plot** — Diagrama de posició d’escenari.

**Mode Productor** — Gestió completa d’events (congelat).
