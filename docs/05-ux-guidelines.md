# UX Guidelines — Estil Kinosonik

## Estil visual
- Mode fosc per defecte
- Liquid-glass amb blur i saturació
- Tipografia: Inter / SF Pro
- Paleta:
  - Accent: #B19240 (daurat)
  - Muted: grisos bluish
  - Veil: vel translúcid

## Components
- Cards amb cantonades suaus
- Breadcrumb `.bc-kinosonik`
- Botons petits (`btn-sm`)
- Avisos amb border-left 3px accent

## Pantalles clau
- Llistat de riders
- Detall de rider
- Anàlisi IA en temps real
- Panell admin

## Accessibilitat
- Contrastos > AA
- Fallback quan no hi ha blur
- Respectar prefers-reduced-motion
